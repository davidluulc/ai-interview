# 学习总览

这套文档现在只保留少数主线入口，旧的详细记录都放在 `docs/learning/archive/`。

## 推荐阅读顺序

1. `01-项目总讲解.md`
   - 先知道项目整体是什么、解决什么问题、面试时怎么介绍。
2. `02-FastAPI后端.md`
   - 学接口、路由、schema、数据库模型、鉴权、日志和测试。
3. `03-RAG工程化.md`
   - 学三个 RAG、文档管理、检索增强、质量评估和可观测性。
4. `04-Agent工程化.md`
   - 学 Agent State、Tool Calls、Decision、fallback、日志和后续 LangGraph 迁移。
5. `05-部署与中间件.md`
   - 学 Docker、Nginx、数据库、Redis、云服务器和上线链路。

## 怎么复习

先看主线文档。遇到不懂的细节，再去 `archive/` 查旧文档。

不要从 archive 第一篇开始顺序硬啃，那会很累，也不适合面试准备。

## 面试准备目标

你最终要能讲清：

- 这个项目为什么不是简单 AI demo；
- FastAPI 后端怎么分层；
- 三个 RAG 怎么协作；
- Agent 和普通 LLM 调用有什么区别；
- RAG 如何做文档治理、检索增强、质量评估和可观测性；
- 如果上线，Docker、Nginx、数据库和后端怎么串起来。
