# 学习文档归档说明

这里保存的是项目开发过程中逐步写下来的详细学习记录。

这些文档没有删除，是为了保留细节和历史上下文。

但日常复习不建议从这里开始看。请先看：

```text
docs/learning/00-学习总览.md
```

当你在主线文档里遇到不懂的细节时，再回到 archive 查对应主题。

常见查找方式：

- 想查 RAG 质量评估：看包含 `RAG质量`、`HitK`、`召回` 的文件。
- 想查 Agent：看包含 `Agent`、`weakTag`、`LangGraph` 的文件。
- 想查部署：看包含 `上线`、`部署` 的文件。
- 想查项目讲解：看 `09-AI模拟面试系统阶段性项目讲解.md`。
