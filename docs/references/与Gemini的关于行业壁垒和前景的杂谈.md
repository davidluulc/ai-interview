# AI 应用开发与后端八股文学习种子

## 一、实习阶段的基础要求

### 1. 数据结构与算法

找实习阶段，算法不需要学到竞赛水平，但需要达到下面的程度：

- 能分析时间复杂度。
- 熟悉数组、链表、哈希表、栈、队列、树。
- 会二分、双指针、滑动窗口、DFS、BFS。
- 能刷 LeetCode 简单题和常见中等题。

### 2. 计算机网络

网络需要重点掌握：

- HTTP 请求与响应。
- 常见状态码。
- Cookie、Token、Authorization。
- TCP 三次握手与四次挥手。
- 长连接。
- SSE 和 WebSocket 的区别。
- Nginx 反向代理。
- 超时、重试、限流的基本概念。

### 3. 数据库与缓存

数据库需要重点掌握：

- MySQL 索引。
- B+ 树。
- 事务 ACID。
- 事务隔离级别。
- 慢查询。
- Redis 基础数据结构。
- 缓存穿透、缓存击穿、缓存雪崩。

### 4. 当前目标

不需要一口气成为架构师，当前阶段先做到：

- 面试官问基础问题时，能说清楚。
- 面试官问项目问题时，能结合项目落地讲。

## 二、个人学习路线

### 1. 主线

Python 后端 + AI 应用工程化。

### 2. 项目

AI 模拟面试系统。

### 3. 短期目标

- 能讲清楚项目。
- 能过实习面试。

### 4. 中期目标

补齐下面这些后端工程能力：

- FastAPI
- MySQL
- Redis
- Docker
- Nginx

### 5. 长期目标

逐步补齐更底层和更工程化的能力：

- 数据结构与算法
- 计算机网络
- 系统设计
- LangGraph / Agent 工作流

## 三、后续整理方向

后续可以围绕下面几个专题继续整理八股文：

- FastAPI 请求链路
- 用户认证与 JWT
- SQLAlchemy 与数据库表关系
- MySQL 索引与事务
- Redis 缓存与限流
- RAG 检索链路
- Agent State 与 Agent Decision
- Docker + Nginx 部署链路
